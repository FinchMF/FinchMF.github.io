Here is a Python example:

```python
import numpy as np

def entropy(probs):
    probs = np.asarray(probs)
    probs = probs[probs > 0]
    return -(probs * np.log2(probs)).sum()

p = [0.7, 0.2, 0.1]
print(entropy(p))
```

```javascript
function softmax(xs) {
  const maxVal = Math.max(...xs);
  const exps = xs.map(x => Math.exp(x - maxVal));
  const sum = exps.reduce((a, b) => a + b, 0);
  return exps.map(v => v / sum);
}

console.log(softmax([1.2, 0.7, -0.4]));
```
